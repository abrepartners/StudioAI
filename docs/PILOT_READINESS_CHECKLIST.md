# Pilot Readiness Checklist (MVP)

## Org setup

1. Brokerage created with at least two offices
2. Teams configured where needed
3. Users assigned with scoped roles
4. Memberships validated for role coverage

## Preset and policy setup

1. Brokerage default presets created
2. Office scoped presets verified
3. Approval-required and no-approval presets tested
4. Disclosure defaults configured with brokerage wording

## Workflow validation

1. Agent submits valid job with assets
2. Approval-required flow reaches review queue
3. No-approval flow routes directly to processing
4. Invalid transition attempts return typed errors
5. Reject/request-change flows function with required reasons

## Delivery and revisions

1. Before/after assets are linked and versioned
2. Delivery requires output assets
3. Revision request requires reason category
4. Revision cycle count increments correctly

## Reporting and exports

1. Dashboard metrics populate for pilot seed data
2. Filters work by date, office, team, status, edit category
3. CSV exports validate schema and scope isolation

## Security and audit

1. Tenant scope isolation tested across roles
2. RBAC blocks unauthorized actions
3. Audit events emit for all key actions
4. Request ID trace links errors and actions

## Operations readiness

1. Pilot support runbook available
2. Incident rollback plan documented
3. Demo seed data loaded for onboarding
